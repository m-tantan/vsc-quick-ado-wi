# What's Next?

Congratulations! You've completed the walkthrough and know how to create Azure DevOps work items from VS Code.

## Quick Reference

### Available Commands

Access these via the Command Palette (`Ctrl+Shift+P` / `Cmd+Shift+P`):

- **Create Azure DevOps Work Item** - Create a work item from selected code
- **Configure Azure DevOps Settings** - Change organization, project, or area path
- **Quick ADO: Sign In** - Sign in to Azure DevOps
- **Quick ADO: Sign Out** - Sign out from Azure DevOps

### Sign In / Sign Out

- **Sign In**: Run **"Quick ADO: Sign In"** to authenticate with your Microsoft account
- **Sign Out**: Run **"Quick ADO: Sign Out"** to clear your authentication
- **Re-configure**: Run **"Configure Azure DevOps Settings"** to change your organization or project
